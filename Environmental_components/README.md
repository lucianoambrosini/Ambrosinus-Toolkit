## ATk Ladybug Hourly Plot Alpha



<br>

**1)** **[ATk Ladybug Hourly Plot Alpha](https://github.com/lucianoambrosini/Ambrosinus-Toolkit/tree/main/Environmental_components/ATk_LBHourlyPlotAlpha)** is a Grasshopper component developed in GhPy, that extends Ladybug Tools' Hourly Plot functionality with advanced alpha highlighting capabilities. It creates colored plots of hourly environmental data collections with the ability to visually emphasize specific hours of the year by fading non-highlighted periods.
